# Feature: FTN_01_006_01 无人机带电池充电

## Test Case 1

- 测试编号: FTN_01_006_01_01
- 测试内容: 无人机带电池充电-电池电量50%
- 前提条件: 
> 1. 无人机正常，电池完好
> 2. 使用标准 Micro USB 线
> 3. 使用符合 FCC/CE 合规的 USB 充电器，规格应为输出 DC 5V/1.5A 及以上
> 4. 保持无人机关机状态
> 5. 保持无人机电池留存50%电量
- 测试步骤:
> 1. 按照入门指南的图示安装无人机电池
> 2. 按压无人机侧边的电源按键
> 3. 等待黄灯慢闪
> 4. 按压无人机侧边的电源按键
> 5. 连接无人机充电线
> 6. 等待无人机充电
- 判定标准:
> 1. PASS: 蓝灯慢闪，正在充电
> 2. FAIL: 其他类型的指示灯闪烁
- 测试结果:
> PASS or FAIL
- 测试清理：
> 1. 若PASS，则等待出现指示灯提示后拔出充电线
> 2. 按照入门指南拔出无人机电池
